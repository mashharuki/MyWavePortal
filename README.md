# MyWavePortal
掲示板機能を実装したdapp用のプロジェクトリポジトリです。

### ローカル環境でイーサリアムネットワークを立ち上げるコマンド
 `npx hardhat node`

### ローカル環境でデプロイ用のスクリプトファイルを実行するコマンド
 `npx hardhat run scripts/deploy.js --network localhost`
